NoPiX for Tailwind

Use arbitrary values with 1rem = 10px

Getting Started

After installation, your NoPiX Tailwind project includes the following subdirectories and files:

  • src/: Contains configuration and style files.
  • src/input.css: Input file for TailwindCSS.
  • src/output.css: CSS file generated by TailwindCSS.
  • src/nopix_theme.css: File containing theme variables (e.g., --1rem: 10px).
  • src/nopix_prefix.css: File containing generated utilities.
  • src/nopix_script.js: Script to synchronize HTML classes with CSS files.
  • src/prefixes.js: File defining supported prefixes and properties.

Step-by-Step Guide

Follow these steps to set up your NoPiX Tailwind project with Node.js:

  1. Ensure Node.js is installed on your machine. Download it from nodejs.org if needed.
  2. Open a terminal in your project directory (where package.json is located).
  3. Install dependencies by running:
    npm install
  4. Run the generation and watch script with:
    npm run nopix
  5. Modify your HTML files (e.g., welcome.html or any HTML file in any directory) to add NoPiX Tailwind classes (e.g., text-3rem, p-1.7rem). Important: Ensure the <link> tag points to src/output.css. For files at the root (e.g., welcome.html), use <link href="src/output.css" rel="stylesheet">. For files in subdirectories (e.g., components/index.html), use <link href="../src/output.css" rel="stylesheet">. With Vite, use <link href="/src/output.css" rel="stylesheet">. Always adjust the path based on the file's location!
  6. The nopix_theme.css and nopix_prefix.css files will automatically update on each modification.
  7. Open your HTML files in a browser (preferably via a server like Vite) to see the results.
